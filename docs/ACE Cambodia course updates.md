## Checklist for unit updates

There are three main tasks to carry out. These should be relatively quick and simple to complete.

Currently our main goal is to get the units into a usable state, removing incorrect references or obvious issues. In some cases this may require you to remove parts of the course that are included as tools for the students to use in MyLearning. Currently no Cambodian version of these tools exist, so these references need to be removed. Ideally these tools will be re-added later in a separate project.


> [!attention] If you are unsure what to do about an element of a unit, please ask for advice before you make a change.

### 1. Update all references to unit codes

This involves removing the “AUR” prefix from unit codes and replacing with “INT”.

The main places to update this are:

- In the first topic (Welcome and getting started)
- In the unit settings (you can also remove the “Development” suffix from the course title here)

If there are documents in the units you will need to check them to make sure the unit code is updated.

Electude documents are not editable, so no need to check these.

### 2. Remove references to MyLearning or Aus campuses

All references to MyLearning or to learners attending classes in Australia need to be removed. This includes the words as well as references to parts of the units that are stored on MyLearning such as Student handbook links.

For each instance of these appearing you will need to decide what is the best course of action.

For example, if there is a reference to MyLearning you may need to:

- Remove the reference and surrounding sentences completely
- Remove the word MyLearning and leave everything else intact
- Replace the word with something more generic such as “LMS”.

### 3. Check units for display issues that can be easily resolved

When you review at the units please look for any small visual issues that can be easily fixed.

This could include:

- Typos
- HTML issues causing text to display incorrectly (for example, font size issues are very common)
- Extra blocks that are empty, see screenshots below.

These are empty “Text and media area” blocks that serve no purpose. They also show up in the main navigation for the unit, as shown. Please remove these.
